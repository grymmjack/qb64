...::: Installation und Starten der QB-MonsterFAQ
Die QB-MonsterFAQ ist bequem mit dem beiliegenden Datenbankprogramm CUEcards 
lesbar. Die QB-MonsterFAQ benötigt keinerlei spezielle Installation und keine 
Einträge in der Windows Registry.

Kopiere einfach alle Dateien des Download-Archivs in ein beliebiges Verzeichnis.
Starte dort CUECARDS.EXE und öffne die QB-MonsterFAQ über "Datei -> öffnen ->
QBmonFAQ.cue"

Über "Extras -> Optionen -> Allgemein -> Automatisch öffnen: QBMonFAQ.cue" 
kannst Du erreichen, dass die QB-MonsterFAQ bei jedem Start automatisch geöffnet 
wird.

Über 417 der in der FAQ aufgeführten Beispielprogramme findest Du als direkt
startbare BAS-Dateien im Verzeichnis PROGS\ . Die HTML-Datei PROGS\_INHALT.HTM
enthält eine nach Rubriken geordnete Liste der Beispielprogramme. Über die
dort angebotenen Links können die Beispielprogramme gestartet oder umgespeichert
werden.  


...::: Wem gehört die FAQ? 
Diese FAQ ist geistiges Eigentum der gesamten deutschsprachigen QBasic-
Community. Der Datenbestand der QB-MonsterFAQ selbst (Datei QBMONFAQ.CUE und 
Beispielprogramme) steht unter einer "Free-Document-Lizenz", die im Eintrag 
"Lizenzbedingungen" in der Kategorie "Hilfe" spezifiziert ist. Unter den dort 
genannten Bedingungen kann die FAQ Datei beliebig modifiziert, erweitert, 
vervielfältigt und weitergegeben werden.


...::: Online-Version der FAQ
Unter http://www.qbasic.de/faq gibt es auch eine Online-Version dieser FAQ.

...::: How to install and launch the QB-MonsterFAQ?
The QB-MosterFAQ is easy to use by means of the provided database programm 
CUEcards. The QB-MonsterFAQ is ready-to-run. There is not need of any special 
installation procedure and Windows registry entries.

Simply copy all files of the downloaded archive into any directory, launch
CUECARDS.EXE and open the QB-MonsterFAQ via  "Datei -> öffnen -> QBmonFAQ.cue" .

Use the "Extras -> Optionen -> Allgemein -> Automatisch öffnen: QBMonFAQ.cue"
option to make QBMONFAQ.CUE the standard start file.

Over 417 of the example programs quoted in the FAQ are available as ready-to-run 
BAS files in the PROGS\ directory. A task-oriented list of all programs with 
short descriptions is provided under PROGS\_INHALT.HTM.

...::: The online version of the FAQ
There is an online version of the QB-MonsterFAQ available on 
http://www.qbasic.de/faq . The online version is only a makeshift HTML export of 
the offline version, missing many of the formatting features. E.g., clickable 
links, multiple blanks and multi linefeeds don't work and it lacks the 100+ 
example BAS-programms. It's heavily recommended to use this downloaded offline 
version and not the online version.


...::: Why has the FAQ been made? 
This FAQ should answer nearly all standard questions on QBasic which are posted 
over and over again on all QBasic boards. Please consult this FAQ before 
annoying the community of any QBasic newsgroup or discussion board with 
alsways the same "stereotype" beginner's question on QBasic.
